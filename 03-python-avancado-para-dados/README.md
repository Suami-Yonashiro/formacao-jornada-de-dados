# Python Avançado para Dados

## 📋 Sobre

Este bootcamp é um **intensivo completo de Python** focado em engenharia de dados. Você vai do zero até tópicos avançados (APIs, ORMs, ETLs) para resolver problemas reais de dados.

**Objetivo Educacional**: Dominar Python aplicado a engenharia de dados, desde fundamentos até construção de APIs e pipelines completas.

## 🎯 Objetivos de Aprendizado

- **Fundamentos de Python**: Sintaxe, tipos, estruturas de dados
- **Programação Orientada a Objetos**: Classes, herança, polimorfismo
- **APIs REST**: Construir APIs com FastAPI
- **ORM e Bancos de Dados**: SQLAlchemy, SQLModel
- **ETL Pipelines**: Construir pipelines de dados
- **Tratamento de Erros**: Logging, exceções, retry
- **Testes**: Testes unitários e de integração

## 📚 Conteúdo Detalhado

Este bootcamp é dividido em **4 fases progressivas**, totalizando **20 aulas práticas** que vão do zero ao avançado:

### Fase 1: Fundamentos (Aulas 01-06)

**Objetivo**: Construir base sólida em Python aplicado a dados

- **Aula 01**: Configuração do ambiente (Python, Git, VSCode) e primeiros passos
- **Aula 02**: Tratamento de erros, type checking e conversão de tipos
- **Aula 03**: Controle de fluxo (if/else, loops), listas e dicionários
- **Aula 04**: Tipos complexos, Type Hints e comparação entre estruturas de dados (Dicionários vs DataFrames vs Tabelas vs Excel)
- **Aula 05**: Projeto prático - Leitura e escrita de arquivos, processamento de grandes volumes (1 bilhão de linhas)
- **Aula 06**: Exercícios de revisão e consolidação dos conceitos

**Resultado**: Você será capaz de escrever scripts Python funcionais para manipulação básica de dados.

### Fase 2: Funções e Estruturas (Aulas 07-10)

**Objetivo**: Dominar funções, modularização e estruturas de dados avançadas

- **Aula 07-09**: Funções em Python (3 partes)
  - Definição de funções, parâmetros, retornos
  - Funções lambda e funções de ordem superior
  - Decoradores e closures
  - Estruturas de dados avançadas (sets, tuples, namedtuples)
- **Aula 10**: Revisão e exercícios práticos

**Resultado**: Você criará código modular, reutilizável e bem estruturado.

### Fase 3: Programação Orientada a Objetos (Aulas 11-16)

**Objetivo**: Aprender POO e aplicar em projetos de dados

- **Aula 11**: Introdução à Programação Orientada a Objetos
- **Aula 12-15**: Classes em Python (4 partes)
  - Criação de classes, atributos e métodos
  - Encapsulamento (private, protected, public)
  - Herança e composição
  - Polimorfismo e métodos especiais
  - Getters, setters e properties
- **Aula 16**: Revisão de POO + SQLModel (ORM moderno)

**Resultado**: Você projetará soluções orientadas a objetos para problemas complexos de dados.

### Fase 4: APIs e Projetos Completos (Aulas 17-20)

**Objetivo**: Construir aplicações completas de dados

- **Aula 17**: SQLAlchemy - Manipulação de SQL em Python
  - Conexões com bancos de dados
  - Models e mapeamento objeto-relacional
  - Queries avançadas
- **Aula 18**: Introdução a APIs REST
  - Conceitos de HTTP, REST, JSON
  - Requests e consumo de APIs
  - Pydantic para validação de dados
  - Implementação de CRUD básico
- **Aula 19**: Criando sua primeira API
  - FastAPI framework
  - Endpoints e rotas
  - Documentação automática
- **Aula 20**: Projeto completo CRUD
  - Backend completo com FastAPI
  - Frontend com Streamlit
  - Integração com banco de dados PostgreSQL
  - Deploy com Docker

**Resultado**: Você construirá aplicações completas de dados prontas para produção.
